# Savings Quest - 貯蓄クエスト

「お金をためるゲームを遊びながら」資産形成の第一歩を踏み出せるビジュアル付き貯蓄トラッカー。

## 特徴

- 🎮 **ゲーム要素**: 貯蓄額に応じてアバターが成長（ひよこ→にわとり→りゅう→かね持ち）
- 📊 **貯蓄履歴**: 棒グラフで週間・月間 savings を視覚化
- 🏆 **バッジシステム**: 達成目標に合わせてバッジを獲得
- 🌙 **ダークモード対応**
- 📱 **PWA対応**: スマホande использование

## 使い方

1. 目標金額を設定
2. 貯蓄を追加（手動入力 or クイックボタン）
3. アバターが成長各地的
4. バッジを獲得して継続のモチベーションに

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- LocalStorage でデータ保存
- PWA (Service Worker)
